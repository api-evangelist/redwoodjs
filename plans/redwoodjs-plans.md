# RedwoodJS Plans

RedwoodJS is an open-source, MIT-licensed full-stack JavaScript framework. There are no commercial subscription tiers for the core framework itself — it is free to use for any project.

## Open Source (Free)

- Full access to the RedwoodJS framework and all core packages
- GraphQL API layer powered by GraphQL Yoga and Envelop
- Prisma ORM integration with automatic SDL generation
- React front-end with Apollo Client pre-configured
- Authentication helpers (dbAuth, Auth0, Clerk, Supabase, and more)
- CLI tooling (`yarn rw`) for code generation, migrations, and deploys
- Storybook and Jest integrations included
- Active community on Discord and community forums
- Deploy targets: Netlify, Vercel, Render, Fly.io, AWS, Baremetal

## RedwoodSDK (rwsdk.com)

A newer Cloudflare-native variant of Redwood (also open source) focused on server-first React with React Server Components running on the Cloudflare platform. Free and MIT licensed.

- Server-first React with React Server Components
- Type-safe routing and SQL (via Cloudflare D1)
- Real-time state sync (`useSyncedState`)
- Deploy exclusively to Cloudflare Workers

## Support & Services

Paid support, consulting, and sponsorship options are available through the project's GitHub Sponsors page and through third-party agencies in the RedwoodJS ecosystem. No official paid SLA tier exists from the core team as of 2026.
